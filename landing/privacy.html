隐私政策

生效日期：2026 年 7 月 3 日

一句话总结：FocusMic 完全在你的 Mac 上本地运行，不收集、不存储、不传输任何个人信息，没有任何网络请求、统计分析或第三方 SDK。

1. 我们收集哪些信息

不收集任何信息。FocusMic 没有账号系统，不采集设备信息、使用数据或任何形式的个人身份信息，也不包含任何统计、埋点或崩溃上报组件。

2. 本地存储的数据

为了记住你的偏好和最近活动，FocusMic 仅在你的 Mac 本地（macOS 标准的 UserDefaults）保存以下数据：

  • 首选输入设备的标识符（UID）；
  • 首选输入设备的名称（用于设备标识符变化时的匹配）；
  • 「守护输入设备」开关的状态。
  • 最近的活动日志（最多 50 条）。

这些数据仅存在于你的设备上，不会被上传到任何服务器。卸载应用后即随之删除。

设置窗口中的「活动日志」会保存在你的 Mac 本地（最多 50 条），用于在应用重启后继续查看最近的设备切换记录。

3. 麦克风与音频

FocusMic 不访问、不录制、不监听任何音频内容。它仅通过 macOS 的 Core Audio 框架读取和设置「系统默认输入设备」这一项系统配置，从不接触麦克风采集的声音数据，因此也不需要麦克风权限。

4. 网络访问

FocusMic 不发起任何网络请求。应用内「关于」面板中的外部链接（如作者主页）仅在你主动点击时由系统浏览器打开。

5. 第三方服务

FocusMic 不集成任何第三方 SDK、分析工具或广告服务。开机自启动功能使用 macOS 系统自带的 ServiceManagement 框架实现。

6. 儿童隐私

由于 FocusMic 不收集任何数据，因此也不会收集儿童的个人信息。

7. 政策变更

如果本政策发生变更，我们会在本页面更新并调整生效日期。由于应用本身不联网，变更不会影响你已安装版本的行为。

8. 联系我们

如对本政策有任何疑问，欢迎通过 GitHub Issues 与我们联系。FocusMic 是开源软件，所有代码均可公开审计。
